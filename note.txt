# BIBLIO READLINE

- readline
        est une fonction puissante qui fait plusieurs choses :
        Affichage du prompt :
        Elle affiche le texte que vous lui donnez en argument ("matteoshell$" dans votre cas)
        Elle attend que l'utilisateur tape quelque chose
        Elle gère automatiquement le curseur
        Gestion de l'entrée :
        Elle lit tout ce que l'utilisateur tape jusqu'à ce qu'il appuie sur Entrée
        Elle retourne une chaîne de caractères (char ) contenant ce que l'utilisateur a tapé
        Si l'utilisateur fait Ctrl+D, elle retourne NULL
        Fonctionnalités intégrées :
        Navigation avec les flèches gauche/droite
        Suppression de caractères avec Backspace
        Copier/Coller
        Historique (avec add_history)
        Exemple concret :
        Apply to note.txt
        ;
        Si l'utilisateur tape "ls" et appuie sur Entrée : line contiendra "ls"
        Si l'utilisateur fait Ctrl+D : line sera NULL
        Si l'utilisateur appuie juste sur Entrée : line contiendra "" (chaîne vide)

- add_history
        add_history est une fonction de la bibliothèque readline qui permet de sauvegarder les commandes dans 
        l'historique du shell. C'est ce qui permet de :
        - Utiliser la flèche du haut (↑) pour voir les commandes précédentes
        - Utiliser la flèche du bas (↓) pour revenir aux commandes plus récentes
        - Avoir un historique des commandes comme dans bash
